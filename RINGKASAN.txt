🔧 CARA MENGGUNAKAN WEBSITE
═══════════════════════════════════════════════════════════════════════════════════

MENJALANKAN DEVELOPMENT SERVER
─────────────────────────────────────────────────────────────────────────────────
  1. Buka terminal/command prompt
  2. Masuk folder: cd /vercel/share/v0-project
  3. Jalankan: pnpm dev
  4. Buka browser: http://localhost:3000

EDITING KONTEN
─────────────────────────────────────────────────────────────────────────────────
  Baca file PANDUAN.md untuk panduan lengkap tentang:
  • Mengubah data desa
  • Menambah/mengubah produk
  • Mengubah teks & konten
  • Mengubah warna & styling

CUSTOMIZATION CEPAT
─────────────────────────────────────────────────────────────────────────────────
  Edit: /lib/constants.ts
  Ubah: Nama desa, nomor telepon, email, jam operasional

  Edit: /app/produk/page.tsx
  Ubah: Produk, harga, deskripsi, kategori

❓ PERTANYAAN UMUM
═══════════════════════════════════════════════════════════════════════════════════

Q: Bagaimana cara mengubah warna website?
A: Edit /app/globals.css - ubah nilai --primary untuk warna utama

Q: Bagaimana cara menambah produk baru?
A: Edit /app/produk/page.tsx - tambah item di array products

Q: Bagaimana form kontak bekerja?
A: Form tersimpan di state React. Untuk email notif, integrasikan backend.

Q: Bagaimana cara deploy ke internet?
A: Gunakan Vercel, Netlify, atau platform hosting lainnya.

Q: Apakah sudah mobile-friendly?
A: Ya! Sudah teruji di mobile, tablet, & desktop.

Lihat PANDUAN.md untuk FAQ lengkap!
